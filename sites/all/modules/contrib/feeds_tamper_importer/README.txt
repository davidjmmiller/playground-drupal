Feeds Tamper Import
===================

Feeds tamper plugin import for Drupal.

Requirements
=============

- CTools 1.x
  http://drupal.org/project/ctools
- Job Scheduler
  http://drupal.org/project/job_scheduler
- Drupal 7.x
  http://drupal.org/project/drupal
- Feeds Tamper
  https://www.drupal.org/project/feeds_tamper
- PHP safe mode is not supported,
  depending on your Feeds Importer configuration
  safe mode may cause no problems though.

Installation
============

- Install Feeds, Feeds Admin UI, Feeds Tamper.
- Go to admin/structure/feeds/your_feed_importer_name/tamper/import
  to import feeds tamper plugin data.
- Paste the exported feeds tamper data into import
  data field and press Import button.
- You can view the created tamper plugins at the
  tamper page of your feeds importer.
